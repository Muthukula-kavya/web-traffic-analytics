# Web Traffic Analytics

## Project Overview

This project analyzes global website traffic and engagement metrics to understand user behavior across different websites. The analysis focuses on key metrics such as monthly visits, bounce rate, session duration, website categories, and user engagement.

## Objective

- Analyze website traffic data.
- Identify the most visited websites.
- Study bounce rates and user engagement.
- Examine session duration patterns.
- Explore website categories and market distribution.
- Generate insights using data visualization.

## Technologies Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Dataset

The dataset contains global website traffic and engagement information, including:

- Global Rank
- Domain Name
- Website Category
- Primary Market
- Monthly Visits
- Bounce Rate (%)
- Average Session Duration (seconds)
- Stickiness Index

## Project Steps

1. Data Collection
2. Data Loading and Exploration
3. Data Cleaning
4. Traffic Analysis
5. Bounce Rate Analysis
6. Session Duration Analysis
7. Category Analysis
8. Data Visualization
9. Findings and Conclusions

## Key Findings

- Top-ranked websites receive significantly higher monthly traffic.
- Bounce rates vary across websites and categories.
- Session duration provides insights into user engagement.
- Social, News, and Finance websites attract large audiences.
- Different countries contribute substantially to global web traffic.

## Visualizations

The project includes:

- Top 10 Websites by Monthly Visits
- Bounce Rate Distribution
- Session Duration Distribution
- Website Category Analysis
- Primary Market Analysis
